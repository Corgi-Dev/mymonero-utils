# Info

## Installing

Using npm:

```bash
$ npm install @mymonero/mymonero-bigint
```

## Example

```js
const JSBigInt = require('@mymonero/mymonero-bigint').BigInteger;
const amount = new JSBigInt('12300000')
const amount_str = monero_amount_format_utils.formatMoney(amount)
```
